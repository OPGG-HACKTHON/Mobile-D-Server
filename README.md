# # JoinUs
*****
## # 프로젝트 소개
- 개발기간 : 2021.07 ~ 2021.09
- 개발언어 : Android(Kotlin), IOS(Swift), Server(SpringFramework, Java)
- 제 1회 OP.GG 정보통신 써-비스 제작 경진대회 참가

기능
- 온보딩을 통한 게이머 맞춤 매칭 서비스 제공
- 매칭 생성, 참여
- 상호 평가를 통한 매너도 제공
- 친구, 블랙리스트 관리
